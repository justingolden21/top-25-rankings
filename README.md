# Top 25 College Football Rankings Website

Currently live at https://sportstop25.netlify.app

Made by [Justin Golden](https://justingolden.me) with Tailwind and Svelte Kit

### Setup

`npm i`

### Dev

`npm run dev` or `npm run dev --open`

### Build and Preview

`npm run build`

`npm run preview`
